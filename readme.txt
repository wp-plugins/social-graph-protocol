The Facebook Open Graph Protocol plugin developed by bizzyLabs enables you to integrate your websites content into the Facebook Open Graph.

== Description ==

The Facebook Open Graph Protocol plugin developed by bizzyLabs is a quick and easy way for you to integrate all of your websites content into the open graph developed by Facebook.
The plugin can be used as simple as activating it and forgetting it or it can be used for more adavnced purposes such as adding Video, Audio, and Location data to specific posts that you write.
Additionally the plugin supports custom post types so that you can fully integrate the social graph in all aspects of your website.

This plugin also enables you to integrate Facebook insights data with your website and provides you with very detailed information on how to do it, along with very detailed information related
to how the open graph works and the benefits of using the open graph.

For a complete write up on the plugin visit the [Social Graph Protocol](http://www.bizzylabs.com/facebook-open-graph-plugin/ "Social Graph Protocol") Homepage

== Installation ==

Installation is made as simple as possible just follow the steps below.

1. Download the zip file and save it somewhere you’ll remember or skip this step and search for "Social Graph Protocol" from the add plugins screen inside your Wordpress admin panel.

2. Jump into your WordPress admin panel and navigate to the Plugins menu.

3. Click the “Add New” button near the top of that page.

4. Near the top click the upload link.

5. Click choose file and select the open-graph-protocol zip file, then press the install now button.

6. Click Activate Plugin.

This is just a very brief installation write up for a much more detailed write up with images and extra information visit the [Social Graph Protocol](http://www.bizzylabs.com/facebook-open-graph-plugin/ "Social Graph Protocol") Homepage
